## 快速上手

### 克隆代码
```
git clone git@github.com:zjunlp/DeepKE.git
```
### 使用pip安装

首先创建python虚拟环境，再进入虚拟环境

然后DeepKE可以依此安装：

```
pip install deepke
```

### 使用工具
先进行训练(训练所用到参数都在conf文件夹中，修改即可)
```
python run.py
```
再进行预测(需进入conf文件夹中predict.yaml修改使用模型路径)
```
python predict.py
```
